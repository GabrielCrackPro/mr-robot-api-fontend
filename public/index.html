This API contains detailed information about the Mr robot tv series.
It allows you to use differents endpoints to GET data such as characters, seasons, episodes an more

Read the documentation for more information.

You can test the api using the form below

The endpoints are:

  • / - GET series details
  • /characters - GET all characters
  • /characters/:id - GET specific character by id
  • /seasons- GET all seasons
  • /seasons/:id - GET specific season by id
  • /episodes - GET all episodes
  • /episodes/:id - GET specific episode by id
  • /episodes/:season/:episode - GET a specific episode by its season
